# Steam-Profile-Loader
이 프로그램은 스팀 프로필 파일을 불러오는것 입니다. 만약 그 유저가 액자를 사용중이면 액자가 합쳐져서 저장됩니다.

url 값에다가 유저 프로필 URL를 대입해주시기 바랍니다.
ex.) https://steamcommunity.com/profiles/{profile_ID}
